# 프로젝트 명 : 잘먹었습니다.

## 1. 프로젝트 소개
음식물 쓰레기로 발생하는 환경오염을 줄이기 위한 프로젝트. 사용자가 음식을 다 먹은 후 빈 그릇을 사진 찍고 인증하면 에코 포인트가 발행된다. 사용자는 획득한 에코포인트로 다양한 기업들의 사회공헌 프로젝트에 기부할 수 있다. 기업들은 프로젝트 참여를 통해 홍보효과를 누릴 수 있다.

- 프로젝트 선정한 배경. (PPT의 내용이 들어가면 좋을 듯)

- PPT 자료 첨부
